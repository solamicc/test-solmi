내가 이해한 개발 기초

수업에서 배운 HTML 태그와 웹 기본 개념을 정리했다.

1. HTML이란?

HTML은 웹 페이지의 구조를 만드는 언어이다. 제목, 문단, 이미지, 버튼 같은 요소를 화면에 배치할 수 있다.

수업에서 배운 태그

  • h1 ~ h4 : 제목
  • p : 문단
  • img : 이미지
  • input : 입력창
  • button : 버튼
  • ul, li : 목록
  • a : 링크

2. 제목과 문단

이건 h3 제목이다

이건 p 태그로 만든 문단이다.

이건 h4 제목이다

제목 태그는 숫자가 커질수록 글자가 작아진다.

3. 이미지

img 태그는 src 속성으로 이미지 주소를 넣는다.

4. 입력창과 버튼

input 태그는 사용자에게 값을 입력받을 때 사용한다.



버튼을 눌러도 아직 동작은 없고, 화면에 보이도록만 만든 상태이다.

5. 링크

a 태그는 다른 웹 페이지로 이동할 수 있게 해준다.
