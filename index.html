내가 이해한 개발 기초 정리

HTML / CSS / JavaScript / 웹 기본 개념을 내가 이해한 방식으로 정리

1) HTML이란?

웹 페이지의 구조(뼈대)를 만드는 언어. 제목, 문단, 이미지, 버튼 같은 “요소”를 배치한다.

  • <h1> : 큰 제목
  • <p> : 문단
  • <img> : 이미지
  • <a> : 링크
  • <button> : 버튼

예시

<h1>안녕하세요</h1>
<p>이건 문단입니다</p>
<button>클릭</button>

2) CSS란?

HTML로 만든 구조에 디자인(스타일)을 입히는 언어. 글자 크기, 색, 간격, 정렬 등을 바꾼다.

  • 선택자(Selector): 누구를 꾸밀지 정함 (예: p, .box)
  • 속성(Property): 무엇을 바꿀지 (예: color, margin)

예시

p {
  color: #333;
}

.box {
  border: 1px solid #ddd;
  padding: 12px;
}

3) JavaScript란?

웹 페이지에 동작을 넣는 언어. 버튼 클릭, 입력값 처리처럼 사용자의 행동에 반응할 수 있다.

미니 실습: 버튼을 누르면 숫자가 올라감

현재 숫자: 0

※ JS는 “이벤트(클릭)”가 발생했을 때 화면을 바꾸는 식으로 동작한다.

예시 코드(개념)

button.addEventListener("click", function () {
  // 클릭하면 실행
});

4) 웹은 어떻게 동작하나?

웹은 기본적으로 요청(Request)과 응답(Response)으로 동작한다.

  • 사용자가 URL(주소)을 입력한다
  • 브라우저가 서버에 “페이지 주세요” 요청을 보낸다
  • 서버가 HTML/CSS/JS 파일을 응답한다
  • 브라우저가 파일을 조합해서 화면을 만든다

내 컴퓨터에서만 열면(file://) 다른 사람은 못 봄. 공유하려면 웹에 올려서(https://) 주소(URL)를 만들어야 함.

내 결론(한 줄)

HTML은 구조, CSS는 꾸미기, JavaScript는 동작. 웹은 요청/응답으로 파일을 받아서 브라우저가 화면을 만든다.

캡쳐 제출 팁: 제목이랑 1~3번 박스가 같이 보이게 화면을 맞춰서 1장 찍으면 깔끔함.
